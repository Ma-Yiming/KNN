## 总结
KNN算法作为一个非常朴素的算法，是非常基础的。但是通过本次实验来看，KNN算法虽然非常基础，但是并不是说基础就不好，KNN算法作为一个基础的算法也能达到一个非常高的准确率。此外，利用Python的面向对象的特点对KNN模型进行打包之后，KNN的实现是一个非常容易的操作。KNN算法中，K的选择更偏向去实验性，启发性，没有特定的规律而言。
